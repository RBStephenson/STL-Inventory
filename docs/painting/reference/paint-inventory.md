## How to use this file

This is a point-in-time inventory snapshot. **In the app, the Paint Shelf is the live source of truth** for which paints are owned — this file is kept for brand handling notes, special-product cautions, and color descriptions that the shelf rows don't capture.

For Pro Acryl and Army Painter Fanatic/Speedpaint codes, also cross-check [[paint-code-conventions]].
All hex values are approximate — use for swatch color chips only, not color matching.
Hex values marked "not sampled" were not in the original pixel-sampled reference charts.

## OTHER BRANDS (handling-critical)

- **FW Acrylic Inks (Daler-Rowney):** 10 inks — CMY primaries (120/412/675), Black (028), Crimson (513), Payne's Grey (065), White (011), Cool Grey (053), Burnt Umber (223), Sepia (251). Very high pigment, extremely transparent. Thin heavily as glazes; very staining — clean airbrush immediately.
- **Turbo Dork Turboshift:** 4 color-shift metallics + medium. ⚠️ Must apply over black basecoat; 2–3 thin coats; airbrush ~1:1 with their medium.
- **Villainy Ink:** 7-ink set. ⚠️ **ENAMEL — not acrylic.** Apply over sealed acrylic (varnish first); clean with enamel thinner/white spirit; 4–12h dry. Weathering/panel lines.
- **Dirty Down:** Moss / Rust / Verdigris (25ml). Water-based effect products; apply over sealed surface, manipulate while wet; final step before matte varnish.
- **AMMO (MIG) DIO Drybrush:** Chipping, Light Metal, Gun Metal, Brass (40ml). Drybrush-formulated — not airbrush/thinned work.
- **Tamiya:** X27 Clear Red (acrylic gloss; candy red/blood) + 87131 Panel Line Accent Black (⚠️ enamel).
- **Mr. Hobby:** Mr. Dissolved Putty + Mr. Surfacer 1000 (⚠️ lacquer — ventilate, test on plastics).
- **Golden:** Airbrush Medium (16oz).
- **Folk Art:** 20-paint dedicated 1:6 base palette (earth tones, greens, greys, 2 metallics). ⚠️ **Use for bases, not figure skin/clothing** — too little pigment control for fine figure work. Do NOT substitute Pro Acryl for these when a guide calls for base work.
